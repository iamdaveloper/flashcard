# Flashcard Mini-Site / 閃卡學習網站

A simple flashcard application for language learning, built with pure HTML, CSS and JavaScript.  
一個簡單的語言學習閃卡應用程式，使用純HTML、CSS和JavaScript構建。

## Features / 功能特點
- **Review Mode**: View all flashcards with words and definitions  
  **複習模式**: 查看所有閃卡，包含單詞和解釋
- **Quiz Mode**: Test your knowledge with randomized questions  
  **測驗模式**: 隨機出題測試你的知識
- **Local Storage**: All data saved in browser's localStorage  
  **本地存儲**: 所有數據保存在瀏覽器的localStorage中
- **Responsive Design**: Works on mobile and desktop  
  **響應式設計**: 在手機和電腦上都能使用
- **No Dependencies**: Pure vanilla JavaScript, no frameworks  
  **無依賴**: 純原生JavaScript，無需框架

## How to Use / 使用方式
1. **Add Flashcards**:  
   **添加閃卡**:
   - Enter a word and its definition in the input fields  
     在輸入框中輸入單詞和解釋
   - Click "Add" button or press Enter  
     點擊"新增"按鈕或按Enter鍵

2. **Review Flashcards**:  
   **複習閃卡**:
   - All your flashcards will be displayed as cards  
     所有閃卡會以卡片形式顯示
   - Click the ✕ button to delete a card  
     點擊✕按鈕刪除卡片

3. **Take Quiz**:  
   **進行測驗**:
   - Switch to Quiz mode  
     切換到測驗模式
   - You'll see definitions - type the correct word  
     你會看到解釋 - 輸入正確的單詞
   - Get instant feedback and final score  
     立即獲得反饋和最終分數

## Technical Details / 技術細節
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)  
  **前端**: HTML5, CSS3, JavaScript (ES6+)
- **Data Storage**: Browser's localStorage (persists between sessions)  
  **數據存儲**: 瀏覽器localStorage (會話間持久保存)
- **Styling**: BEM methodology for CSS class names  
  **樣式**: 使用BEM方法命名CSS類
- **Responsive**: Flexbox and Grid layout  
  **響應式**: 使用Flexbox和Grid佈局
